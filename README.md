# Project - dramaday.news Blogger Theme 🎬🌸

Welcome to the official repository for **Project - dramaday.news**. This is a premium, high-performance Blogger template custom-engineered for Asian Drama hubs. Designed specifically to power portals covering **K-Drama, J-Drama, and C-Drama**, this theme seamlessly integrates entertainment news publishing, episode download indexing, and subtitle distribution.

It provides a sleek, modern UI/UX crafted to deliver an immersive experience for the Asian drama community.

## 🚀 Key Features

- 📰 **Entertainment News Engine:** Clean grid and list layouts optimized for drama news, casting updates, and celebrity rumors.
- 📥 **Media Download Architecture:** Dedicated UI blocks for multi-resolution episode downloads (480p, 720p, 1080p) and mirror links.
- 📝 **Subtitle Hub Ready:** Structured layouts for SRT/ASS file downloads, sync versions, and translator credits.
- 📱 **Fluid Responsiveness:** 100% responsive design that adapts beautifully to mobile devices, tablets, and desktops.
- 🎨 **Cinematic Dark Aesthetic:** Premium dark theme with vibrant accents designed to maximize user session time.
- ⚡ **SEO & Performance Tuned:** Lightweight XML structure mapped for fast loading speeds and optimal Google Search ranking.

## 🛠️ Installation Guide

1. Open the `theme.xml` file in this repository and copy the entire XML code.
2. Log in to your **Blogger Dashboard**.
3. Go to **Theme** -> click the dropdown arrow (🔽) next to *Customize* -> select **Edit HTML**.
4. Select all existing code, delete it, paste the new XML code, and click **Save**.

*⚠️ Pro-Tip: Always create a backup of your existing theme before replacing the XML code.*
